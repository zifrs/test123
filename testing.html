This is a paragraph tag, which means you can write a paragraph and the chunk of css in the head section will dictate what it looks like. Similarly, you can create whatever tag you want and it'll act as a sort of "link" between your css style and what shows up in the html. The "p" tag is so-called "standard" html/css, but you can make a tag that's called "myspecialtag", and if it's between angled brackets in the html body section, it'll obey whatever css you describe for it in the head section.
